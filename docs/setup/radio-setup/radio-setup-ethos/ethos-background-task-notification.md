# FrSky Ethos Background Task Notification

Ethos Background Task Notification

On a new installation of the Ethos Suite you may be presented with the error enable background task.

![Ethos](/rotorflight-docs/assets/images/ethos-background-1-a5f59e78efc249099fd550d8edf06df0.jpg)

Press the \[MDL} button and scroll to the LUA icon and press enter

![Ethos](/rotorflight-docs/assets/images/ethos-background-2-e191cec5b9744802970a7ceedfe0c9b3.jpg)

Enable the BG tasks and this will fix the error

![Ethos](/rotorflight-docs/assets/images/ethos-background-3-d335d84ef96a6e122ed6bea343952e77.jpg)
