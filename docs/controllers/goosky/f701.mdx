# Goosky F701-ELRS

Designed specifically for Rotorflight, the Goosky F701-ELRS provides fast response and precise, stable control for RC helicopter dynamics.

![Goosky F4-mini](../img/goosky-f701.jpg)

:::info[Specifications]
### Goosky F701-ELRS Rotorflight Flybarless Unit with built-in ELRS Receiver

- Main MCU: STM32F722RET6 
- Gyroscope / Inertial Measurement Unit (IMU): ICM42688P 
- Barometer: SPLC6 
- Blackbox flash memory: 256MB 
- Firmware: Rotorflight 
- Servo ports: 4: CH1, CH2, CH3, TAIL 
- Supported receiver protocol: CRSF, ELRS, S.BUS, FBUS, PPM, MSP, DSMX 
- Supported ESC protocol: PWM, Oneshot, Multishot, DShot300, DShot600 
- Motorized tail: supported 
- RPM sensor: supported 
- UART ports: 6 
- Input voltage range: 5–16V 
- Serial port output power: 5V / 1.5A (ensure connected devices do not exceed this current) 
- Built-in ELRS receiver using UART1 
- Unit size: 46 × 26 × 14.5 mm; 22 g 
- Package Size: 9 × 6 × 3.5 cm; 84 g 
:::

### Rotorflight Target

Please use the *GOOSKY\_F701* target when updating Rotorflight firmware.
